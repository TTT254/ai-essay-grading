## 🚀 使用步骤

### 方式1：直接在前端注册（推荐）

1. 访问 http://localhost:5173/register
2. 依次注册上述账号
3. 注册完成后，在 Supabase SQL Editor 执行 `docs/test-data.sql`

### 方式2：使用 SQL 导入数据

1. 先在 Supabase SQL Editor 执行 `docs/test-data.sql` 创建班级和作业
2. 在前端使用测试账号注册
3. 注册时填写对应的姓名和班级

## 📝 测试数据说明

执行 `test-data.sql` 后会创建：

- ✅ **5个班级** - 7年级1班、7年级2班、8年级1班、8年级2班、9年级1班
- ✅ **3个作业** - 我的家乡、难忘的一天、春天来了
- ✅ **1个作文提交** - 学生1提交的"我的家乡"
- ✅ **1个批改报告** - AI批改的示例报告

## ⚠️ 注意事项

1. **用户ID问题**：
   - SQL中的用户ID是示例，实际需要替换为 Supabase Auth 创建的真实UUID
   - 建议先注册账号，然后在数据库中查询真实ID后更新

2. **执行顺序**：
   - 先执行 `supabase-init.sql` 初始化数据库
   - 再执行 `test-data.sql` 导入测试数据
   - 或者先创建班级，再注册账号

3. **班级关联**：
   - 学生注册时需要选择班级
   - 如果班级不存在，请先执行 SQL 创建班级
